Underneath is a table of contents, feel feree to read!

---

## Table of contents
- Left sidebar with a main content section - displays the content
- Window control buttons: X to close, - to minimize
- Polygon section: Calculates the dimensions required for a polygon with the desired number of sides and radius!
- Bridging blocks section: Bridge blocks (Connect blocks) with any X and Y coordinate
- Pyramid section: Create pyramids with any radius, height, and number of sides
- Settings section (Work In Progress!)

---

## The app's purpose
The purpose of this app is to aid other builders in creating polygons, pyramids, or bridge blocks.
It's basically a third-party build "helper".
